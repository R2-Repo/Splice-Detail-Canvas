# Optional long-form memory

Use this folder for **session notes, decision logs, or research** that belong in git but are too long or too narrative for `TASK_HISTORY.md`.

**Naming:** `YYYY-MM-DD-short-topic.md` or `topic-slug.md`.

**Do not** duplicate every chat message here. Prefer short, factual bullets: decisions, URLs, constraints, and what changed.

If this folder only contains this README until you add notes, that is expected.
