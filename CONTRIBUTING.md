# Contributing

This repository is an **agentic static-frontend framework**: humans and Cursor agents share the same rules in `AGENTS.md` and the same **versioned memory** in docs.

## Before you open a pull request

1. Follow `docs/DEFINITION_OF_DONE.md`.
2. Run `npm run check`, `npm run build`, and `npm run test:ci`.
3. Keep the change **focused**; prefer one concern per PR.
4. Do not merge your own PR if your team uses external review; agents should not self-merge per `AGENTS.md`.

## Issues

Use `.github/ISSUE_TEMPLATE/bug_report.md` when reporting regressions so context (branch, agent, steps) is captured.
